para sincronizar uma pasta criada do repositório 
git clone <endereço em que esta o repositório>. 
Ele sincroniza o repositório na pasta de interesse após a sincronização 
clicar na pasta récem sincronizada
na barra de endereço apagar tudo e digitar cmd após o prompt de comando estar aberto digitar
git init -> inicia o repertório.
git add . -> adiciona todos os arquivos criados git commit -m "comentario" -> adiciona 
um comentario para ser inserido no repositório
git push -> adiciona o arquivo criado ao repositório.
